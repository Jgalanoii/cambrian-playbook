## Architecture

```
React SPA (App.jsx ~10K LOC)
  ├── 4 components (OrgPanel, ReportPanel, SuperAdmin, S9_SolutionFit)
  ├── 23 data files (17 knowledge layers + frameworks + playbooks)
  ├── 2 lib files (supabase.js, utils.js)
  └── App.css (design token system)

Vercel Serverless (6 endpoints)
  ├── api/claude.js (non-streaming Claude proxy)
  ├── api/claude-stream.js (streaming Claude proxy)
  ├── api/knowledge.js (JWT-protected knowledge layer)
  ├── api/admin.js (superuser analytics)
  ├── api/invite.js (team invitations)
  ├── api/cron-reset-tokens.js (monthly reset)
  ├── api/_guard.js (shared auth/security)
  └── api/_usage.js (billing/provisioning)

Supabase (PostgreSQL + Auth)
  ├── 6 tables: users, orgs, sessions, invitations, api_usage_log
  ├── 10 migrations
  ├── RLS on all tables
  └── 3 SQL functions (accept_invitation, increment_run_count, increment_max_run_count)

Knowledge Layer (JWT-protected, never in client bundle)
  ├── 10 scored verticals (Payments, RE, Banking, Healthcare, AI/ML, Fintech, Rewards, QSR, BaaS, Charitable)
  ├── 4 cross-cutting (Accounting, B2B Sales, OKR/KPI, Investor Intelligence)
  ├── 13 compliance frameworks x 10 verticals
  ├── 15 vertical playbooks
  └── 10+ sales methodology frameworks
```
