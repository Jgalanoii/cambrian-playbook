# Discovery Call Scorecard

> An instrument — not a document to read through, but a form to fill out. Built for three use cases: (1) a manager observing a ride-along, (2) a rep self-scoring after a call, (3) a certification evaluator scoring a rep in their 90-day test. All three should produce the same scores on the same call, given the same dimensions. That's the point of a standardized scorecard.

## How to use this scorecard

**Not every call needs to be scored.** Score 1 call per rep per week for new reps (weeks 1–12), 1 call every 2 weeks for established reps, and every certification call. More than that and the scorecard becomes busywork; less than that and you lose calibration.

**Three scoring modes:**

- **Ride-along (live):** manager observes silently, scores in real time. Debrief within 24 hours.
- **Recorded call review:** manager and rep watch together (or separately and compare). Best for asynchronous coaching and calibration.
- **Self-scoring:** rep scores their own call before the 1:1. Manager scores independently. Compare. Discrepancies are the learning.

**Calibration matters.** Multiple managers scoring the same call should produce scores within 0.5 of each other on average. When they don't, that's a calibration exercise — watch the call together, debate dimensions, align.

**Don't coach every dimension at once.** After scoring, pick *one* dimension to coach on per call review. Reps can only improve 1–2 skills at a time.

**Score honestly, not kindly.** A scorecard that inflates scores to protect feelings produces reps who plateau. Honest scoring creates discomfort that drives growth. The rep should know that the scorecard is the truth of the call, not a performance review.

---

## Scoring scale

Every dimension uses the same 1–5 scale:

- **5 — Excellent.** Did this notably well. Worth reinforcing and repeating.
- **4 — Strong.** Consistent with what a senior rep would do.
- **3 — Acceptable.** Met the bar; not notable either way.
- **2 — Weak.** Missed key moments or handled poorly.
- **1 — Failed.** Critical dimension missed entirely or handled in a way that damages the deal.
- **N/A — Not applicable** to this call (e.g., demo skills on a pure discovery call).

**Critical dimensions** (marked ⚠️ below) must score ≥ 2. A score of 1 on any critical dimension = call failed regardless of other scores. For certification, critical dimensions must score ≥ 3.

# Section 1 — Pre-call preparation

Scored before the call starts, based on what the rep brought into the meeting.

| # | Dimension | 1 | 3 | 5 | Score |
|---|---|---|---|---|---|
| 1.1 | **Research on the account** — company, industry, stage, recent news, competitive context | No research | Knows basic firmographics | Has specific, non-Google-able insight ready | |
| 1.2 | **Research on the buyer(s)** — role, tenure, background, public statements or posts | No research | LinkedIn review done | Knows buyer's priorities, past work, recent signal | |
| 1.3 | **Hypothesis set** — rep has a working hypothesis about likely pain and fit before the call | No hypothesis | Vague "they might need X" | Specific CS/FS/Gap hypothesis to test | |
| 1.4 | **Call agenda prepared** — sent in advance or ready to propose | No agenda | Generic discovery agenda | Tailored to prospect's stated interest with clear outcomes | |
| 1.5 | **Meeting mechanics** — on time, camera on, audio good, environment appropriate | Issues | Adequate | Professional presence | |

# Section 2 — Opening & rapport (first 5 minutes)

The opening sets the tone. Done well, the prospect leans in. Done badly, the rest of the call is recovery.

| # | Dimension | 1 | 3 | 5 | Score |
|---|---|---|---|---|---|
| 2.1 | **Framing the meeting** — agenda confirmed, time confirmed, outcomes stated | No framing; dives in | Asks if agenda works | Proposes agenda, invites adjustment, confirms time | |
| 2.2 | **Tone / presence** — warm but not fawning; confident but not salesy | Either stiff or over-performing | Professional | Warm, present, curious | |
| 2.3 | **Permission / consent moves** — asks permission before probing; signals it's a two-way conversation | Doesn't ask | Asks once | Asks strategically; explicitly invites pushback | |
| 2.4 | **Avoids premature pitching** ⚠️ — doesn't present company or product in the first 5 minutes | Pitches in opening | Brief company context if asked | Waits to be invited; keeps opening prospect-focused | |

# Section 3 — Question hierarchy (SPIN lens)

This is the most trainable dimension in a discovery call. Track the mix.

| # | Dimension | 1 | 3 | 5 | Score |
|---|---|---|---|---|---|
| 3.1 | **Situation question discipline** — uses sparingly; does homework first | Over 40% of questions are Situation | 20–30% Situation | ≤ 20% Situation; fact-finding pre-done | |
| 3.2 | **Problem question depth** — surfaces real, specific problems | Generic problem questions | Specific to role/situation | Probing, layered, follow-up-driven | |
| 3.3 | **Implication questions** ⚠️ — builds business impact of problems | None | 1–2 asked | Multiple, building a chain of consequence | |
| 3.4 | **Need-payoff questions** ⚠️ — buyer articulates value in their own words | None | 1 asked | Buyer describes value multiple times unprompted | |
| 3.5 | **Question sequencing** — follows S→P→I→N logic naturally, not mechanically | Random or scripted | Mostly sequenced | Flows as conversation, not framework | |
| 3.6 | **Silence / wait-time** — after asking, lets the prospect think | Fills silences | Occasional silence | Comfortable with 5+ second pauses | |

# Section 4 — Diagnosis (Gap Selling lens)

Can the rep complete a PIC from this call?

| # | Dimension | 1 | 3 | 5 | Score |
|---|---|---|---|---|---|
| 4.1 | **Current State captured** ⚠️ — rep surfaces real, specific, quantified current state | Vague CS | Basic CS | Rich, multi-dimensional CS with quantification | |
| 4.2 | **Future State captured** ⚠️ — rep surfaces what the prospect wants, in their terms | No FS | Generic "improvement" FS | Specific, measurable FS with why | |
| 4.3 | **Gap quantified** ⚠️ — size of gap stated in business terms (revenue, cost, time, risk) | Unquantified | Roughly quantified | Specifically quantified with multiple dimensions | |
| 4.4 | **Root cause probed** — rep asks *why* the problem exists, not just what it is | Never probes root cause | Probes once | Persistent root-cause inquiry | |
| 4.5 | **Business impact beyond the contact** — rep understands consequences for other functions/leaders | Single-function framing | Acknowledges adjacent impact | Maps the pain chain explicitly | |
| 4.6 | **Four Forces identified** (Moesta) — rep has a read on Push, Pull, Anxiety, Habit | No read | Names 1–2 forces | Names all four, identifies the dominant force | |

# Section 5 — Qualification (MEDDPICC lens)

By end of call, what does the rep know?

| # | Dimension | 1 | 3 | 5 | Score |
|---|---|---|---|---|---|
| 5.1 | **Metrics surfaced** — the numbers the prospect will use to measure success | None | Qualitative only | Specific, quantified, in prospect's own terms | |
| 5.2 | **Economic buyer identified** ⚠️ — rep knows who signs, or has a path to finding out | Unknown; not asked | Asked; answer vague | Named; access path mapped | |
| 5.3 | **Decision criteria probed** — what will they use to decide? | Not asked | Asked generically | Probed specifically; written vs. unwritten understood | |
| 5.4 | **Decision process mapped** — steps from today to decision | Unknown | Vague timeline | Specific steps, owners, dates | |
| 5.5 | **Critical Event named** ⚠️ — why must this happen by [date]? | No event | Loose timeline | Specific, externally-driven, confirmed | |
| 5.6 | **Pain quantified** — financial/business cost of inaction | Qualitative pain | Rough quantification | Specific quantification, multiple dimensions | |
| 5.7 | **Champion signal read** — rep has a read on whether this person is a champion or coach | No read | Implicit read | Explicit read with test applied | |
| 5.8 | **Competition surfaced** — what else is being considered? | Not asked | Asked, surface answer | Named alternatives (incl. "do nothing") with understanding of why | |

# Section 6 — Solution mapping

Does the rep move from diagnosis to a specific recommendation?

| 6.2 | **Specific offer referenced** — rep says which of our offers is the fit, or flags that none fits | Generic "our product" | Mentions general category | Names specific offer with rationale | |

# Section 7 — Customer experience behaviors

The CX layer. Easy to miss; high leverage on buyer experience and repeat/referral.

| # | Dimension | 1 | 3 | 5 | Score |
|---|---|---|---|---|---|
| 7.1 | **Active listening** ⚠️ — rep paraphrases back, acknowledges what they heard before responding | Doesn't acknowledge | Occasional paraphrase | Consistent "what I'm hearing is..." moments | |
| 7.2 | **Directiveness (JOLT)** — when buyer is uncertain, rep makes specific recommendations rather than listing options | Over-helpful; lists options | Neutral | Directive and specific ("I'd recommend X for your situation") | |
| 7.3 | **Effort reduction** — rep anticipates the buyer's next question, offers context unprompted | Creates effort | Neutral | Pre-empts buyer's follow-ups | |
| 7.4 | **Anxiety handling** — rep surfaces and addresses unspoken concerns, not just stated ones | Ignores anxieties | Addresses if raised | Surfaces unraised anxieties intentionally | |
| 7.5 | **Pace and rhythm** — call has breath; not a firehose of questions or information | Rushed or rambling | Adequate pacing | Clear rhythm; buyer feels in control | |
| 7.6 | **Honest posture** — rep names trade-offs, acknowledges limitations, doesn't over-promise | Oversells | Neutral | Credibly honest; names real gaps | |

# Section 8 — Close & next steps

How the call ends determines the deal's momentum.

| # | Dimension | 1 | 3 | 5 | Score |
|---|---|---|---|---|---|
| 8.1 | **Specific next step proposed** ⚠️ — concrete action with date, owner, and purpose | "Let's stay in touch" | Vague next meeting | Specific action, date, attendees, outcome | |
| 8.2 | **Commitment test** — rep asks for something specific (an intro, a document, a meeting with EB) | Doesn't ask | Asks for a meeting | Asks for meaningful commitment (intro, data, access) | |
| 8.3 | **Summary of what was heard** — rep summarizes pain, desired outcome, fit read at end | No summary | Brief recap | Complete recap; checks understanding | |
| 8.4 | **Champion enablement (if applicable)** — rep offers the contact something to share internally | Nothing offered | Offers deck/one-pager | Offers tailored kit; asks what would help them sell internally | |
| 8.5 | **Honest fit read** — rep names their assessment of fit before ending | No read given | Generic "good fit" | Specific fit read with both positives and open questions | |

**Weighted average:** total / sections scored (skip N/A).

**Interpretation:**
- **≥ 4.2 weighted avg.** — senior-rep caliber. Exemplar call; worth sharing with the team.
- **3.5–4.1** — solid call. Coach the 1 weakest dimension.
- **2.8–3.4** — developmental call. Significant coaching needed before next deal advance.
- **< 2.8** — call at risk. Don't advance the deal without intervention; re-run discovery.
- **Any critical dimension (⚠️) scoring 1** — call failed regardless of total. Remediate before continuing the deal.

# Red flags (stop-the-call or immediate-intervention behaviors)

Not scored dimensions — these are yes/no flags the manager watches for. Any one observed = note it; multiple = address in same-day debrief, not next 1:1.

- [ ] **Pitched the product in the first 10 minutes.** (Instinct to sell overriding diagnostic discipline.)
- [ ] **Answered a question the buyer didn't ask.** (Showing off features; not listening.)
- [ ] **Took a claim at face value without probing** (e.g., "yes I'm the decision-maker" → moved on without testing).
- [ ] **Disparaged a competitor** by name.
- [ ] **Blurted a specific price** before establishing any fit or context.
- [ ] **Left with a vague next step** ("let's circle back," "I'll be in touch").
- [ ] **Talked more than the prospect.** Target is ≤40% rep talk time in discovery.
- [ ] **Used unnecessary jargon** or acronyms without explaining.
- [ ] **Failed to acknowledge a significant emotional signal** — the buyer expressed frustration, fear, or excitement and the rep moved past it.
- [ ] **Did not ask about the decision process** at all.
- [ ] **Did not identify the economic buyer** or ask who that is.
- [ ] **Over-promised** on capability, timeline, or outcome.
- [ ] **Committed to an action the rep cannot deliver** (e.g., a custom feature, a bespoke pricing move).

---

# Green flags (reinforce, share, replicate)

Behaviors worth highlighting publicly (with the rep's permission) as exemplars of how the team should operate.

- [ ] **Opened with a specific, non-obvious insight** about the prospect's business.
- [ ] **Surfaced an anxiety the buyer hadn't voiced** and addressed it.
- [ ] **Got the buyer to quantify the pain** in dollars, hours, or people.
- [ ] **Disqualified respectfully** when evidence suggested non-fit, offered a partner referral.
- [ ] **Deployed a reference story that landed** — the buyer asked follow-up questions about the reference.
- [ ] **Made a specific directive recommendation** ("I'd suggest Package B for your situation because...").
- [ ] **Ended with a commitment test** — asked for a specific action and got it.
- [ ] **Named the fit honestly** at the end, including open questions.
- [ ] **Anticipated the buyer's next question** and pre-empted it.
- [ ] **Held comfortable silence** after a Need-payoff question; let the buyer articulate value.
- [ ] **Reframed a competitive comparison** on a different axis, without disparagement.
- [ ] **Got a warm introduction** to another stakeholder during the call.

# Post-call debrief framework

For managers running a structured debrief. 30 minutes, within 24 hours of the call.

## Minute 0–5: Rep self-assessment first

Ask: "How did it go? What would you change?"

Don't share your scoring yet. Let the rep assess without priming. This tests their self-awareness — which is itself a trainable skill.

## Minute 5–15: Walk the call together

Play 3 moments back:
1. One moment where the rep did well.
2. One moment where they missed an opportunity.
3. One moment where the buyer's behavior was the signal (e.g., a key phrase, a hesitation, a reveal).

For each: what was happening, what did the rep do, what could they have done?

## Minute 15–25: Share scores

Walk through the scorecard sections. Focus only on:
- The two highest-score dimensions (reinforce).
- The two lowest-score dimensions (improve).
- Any critical-dimension failures (remediate).

Skip mid-range dimensions. The rep can only absorb so much.

## Minute 25–30: One thing

Ask: "If you took one thing from this review into your next call, what would it be?"

Let the rep name it. Write it down. Review at the start of their next call prep.

# Coaching priorities by rep tenure

What to emphasize for reps at different stages.

## First 30 days (new rep)

Prioritize (in order):
1. **Pre-call preparation.** Reps who skip prep fail everything downstream.
2. **Section 4 — Diagnosis.** If they can't diagnose, nothing else matters.
3. **Section 3 — Question hierarchy.** S/P/I/N ratio is the most measurable coaching.

Ignore (for now):
- Nuanced champion development.
- Multi-threading.
- Advanced objection handling.

## Days 30–90 (ramping rep)

Add to above:
1. **Section 5 — Qualification.** MEDDPICC discipline.
2. **Section 6 — Solution mapping.** Getting to specific offer recommendation.
3. **Section 7 — CX behaviors.** Especially directiveness (JOLT) and effort reduction.

## Post-90 days (independent rep)

Shift emphasis:
1. **Section 8 — Close & next steps.** Every call should end with a commitment test.
2. **Section 7.6 — Honest posture.** Senior reps are credible because they're honest; juniors often aren't.
3. **Pattern across calls.** Watch for consistent weaknesses across recordings rather than coaching each call individually.

## Senior reps (year 2+)

- Calibration partner: senior reps score newer reps' calls.
- Mentorship: 1 hour/month pairing with a newer rep.
- Complex deal coaching: high-stakes deals get deep review rather than weekly calls.

# Tracking progress over time

One call's score is a data point. A quarter of calls is a pattern.

## Manager's rep dashboard

For each rep, track:

| Metric | Target | Rep score (this month) | Trend vs. last month |
|---|---|---|---|
| Avg weighted score across scored calls | ≥ 3.5 | | |
| Critical dimension failures | 0 | | |
| Red flags per call (average) | ≤ 1 | | |
| Green flags per call (average) | ≥ 2 | | |
| Self-vs-manager score delta | ≤ 0.5 | | |
| % of calls with specific next step | 100% | | |
| % of calls with EB identified by call 2 | ≥ 80% | | |
| % of calls with Critical Event named by call 2 | ≥ 75% | | |

## What patterns tell you

- **Score plateau at 3.0–3.4 for 4+ weeks** → rep has hit a skill ceiling at current coaching intensity. Change the coaching: new role-plays, different book, direct intervention, or manager-led call.
- **Scores improve but pipeline doesn't close** → scoring well on the wrong things, or scorecard doesn't reflect the real deal drivers in your motion. Recalibrate.
- **Scores regress mid-ramp (week 8–10 commonly)** → normal. Complexity is real, overconfidence from early wins is wearing off. Hold the line.
- **Self-scores consistently higher than manager scores** → rep lacks self-awareness. Work on this explicitly; it blocks development.
- **Self-scores consistently lower than manager scores** → rep lacks confidence or has imposter dynamics. Name it; reinforce observed capability.
- **Green flags plentiful, total score average** → rep has spike skills but base inconsistency. Identify the spike as differentiator; lift the floor.
- **Red flags recurring in same dimension** → systemic habit, not situational miss. Dedicated coaching on that dimension.

# Calibration

The scorecard only works if scorers agree. Run calibration quarterly.

## The quarterly calibration exercise

1. Select one call recording (ideally from a mid-tier rep).
2. Every manager and senior rep scores it independently.
3. Convene and compare. Walk through each dimension.
4. Discuss divergences. What does each person see? What's the "correct" score?
5. Update the scorecard if the team discovers the dimensions need sharpening.

## Calibration standards

- No two managers should score a given dimension more than 1 point apart.
- Average scores across managers should have a standard deviation ≤ 0.5.
- If calibration is off: either the scorecard needs clarification or the managers need training. Address before blaming reps for inconsistent scoring.
